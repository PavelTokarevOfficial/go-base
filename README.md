# Go Base

Учебный проект на Go с PostgreSQL в Docker.

## Запуск

```bash
docker compose up -d
```

Проверить подключение Go-приложения:

```bash
go run main.go
```

## Админка БД

Pgweb доступен в браузере:

http://localhost:8080

Авторизация не нужна: Pgweb автоматически подключается к базе `go_db`.

## Настройки окружения

Основные настройки лежат в `.env`.

Для нового окружения можно скопировать пример:

```bash
cp .env.example .env
```
